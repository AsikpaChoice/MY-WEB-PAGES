WHAT IS HTML?

  • HTML which means hypertext mark-up language is the standard markup language for creating Web pages.
  • HTML defined the structure of a web-page. With HTML, you can create a website.
  • The person who invented HTML IS BERNER'S-LEE .

STRUCTURE OF HTML

< DOCTYPE html >
below is the structure of HTML;
< html >
< head >
< title > Page Title < /title >
< /head >
< body >
<h1> This is the heading </h1>
<p> This is a paragraph </p>
< /body >
< /html >

HTML ELEMENTS

An HTML element is defined by a tag, some content and an end tag. HTML elements tell the browser how to display the content. There are two types of HTML elements; elements with content such as <p>   this is an element with content </p> and empty tags: this tag doesn't have any content in it. e.g <br>
below is the structure of HTML element;
<p> content </p>
